## Criando o projeto

### Comando para listas os comandos do Ionic
```script
ionic start -h
```
### Criando um projeto com o template em branco
```script
ionic start NomeDoMeuProjeto blank
```
### Subindo o Ionic Serve
```script
ionic serve
```
